Overview of Exp 2

Participants complete a rapid serial image-detection task across two blocks:
- block 1 : participants memorize a "Containment" (_C) image and must detect it
- block 2 : participants memorize a "Support" (_S-top) image and must detect it

On each trial, participants see a fixation cross, then a blank screen, then the image, and must press:
- F if the image is the current block's target
- J if it is not

Sequence: 
intro -> language questionnaire -> demographics -> instructions (fullscreen, demo, reminder)
  → block 1 intro -> block 1 trials (96)
  → block 2 intro -> block 2 trials (96)
  → send -> end 

Target Selection: 
- Two image pairs are randomly chosen each run: one as the Containment target (Block 1), one as the Support target (Block 2).
- The complementary condition images for those two pairs (ex. the Support version of the Containment-target pair) are excluded from the non-target pool.
- Non-target trials are filled by cycling through a shuffled pool of the remaining image pairs (both Containment and Support versions) until the block is full.
- Each image is randomly assigned a horizontal-flip orientation per trial.
